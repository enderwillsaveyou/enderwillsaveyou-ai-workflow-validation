# Prompts

This folder stores reusable prompts for starting, bootstrapping, handing off, documenting, and improving AI-assisted projects.

## Prompt Format

Each saved prompt should include:

```markdown
# Prompt Title

## Purpose

## Best Used With

## Inputs Needed

## Prompt

## Expected Output

## Notes

## Version
```
