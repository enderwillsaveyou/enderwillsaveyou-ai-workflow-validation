# Vision

## Problem Being Solved

Important project knowledge often gets trapped in temporary AI conversations. This makes it harder to continue work across ChatGPT, Claude, Gemini, GitHub Copilot, local coding agents, and future tools.

## Long-Term Goal

Create a reusable AI Project Operating System that makes every new project easier to start, document, transfer, and improve.

## Who Benefits

- Mike, as the primary user and project owner.
- Future AI assistants working across different platforms.
- Any future collaborators who need clear repo context.

## What Success Looks Like

A new AI assistant can open the repo, read the instruction files, understand the project status, and safely continue the work without depending on old chat history.
